# Mysql数据库备份与恢复系统

#### 介绍
Mysql数据库备份与恢复系统，实现数据库在线备份、定时备份、数据库恢复、用户管理等功能。

#### 软件架构
基于Spring Boot、Mybatis Plus、Vue.js、ElementUI的前后端分离系统。
